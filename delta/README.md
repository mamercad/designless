# delta Theme

Git diff viewer theme for Designless.

## Installation

Add to `~/.config/delta/themes.gitconfig` or reference from your main `~/.gitconfig`:

```
[include]
  path = ~/.config/delta/themes/designless-light.gitconfig
  # or: path = ~/.config/delta/themes/designless-dark.gitconfig

[delta]
  features = designless-light
  # or: features = designless-dark
```

## Colors

Generated from [`../colors/designless-light.json`](../colors/designless-light.json) and [`../colors/designless-dark.json`](../colors/designless-dark.json).
